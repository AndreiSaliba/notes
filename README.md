# Notes

Notes app made with React, Firebase and Geist UI

![Notes Light Mode](https://i.imgur.com/uhKRtZM.png)
![Notes Dark Mode](https://i.imgur.com/et0qZjG.png)

### Demo Account

Use "demo@example.com" and "password" to login into the demo account.

### Masonry Layout

Set `layout.css.grid-template-masonry-value.enabled` in Firefox's **about:config** to **True** to enable Masonry Layout.
